Solution

Enterprise Information Security

Each host in the enterprise internal network is regarded as a node. The usage rate and connection direction of different network protocols between nodes can be modeled into behavioral characteristics of different probability. The probability of exceeding the probability threshold is regarded as abnormal, and is utilized in the Insider Threat or APT.

Homeland Security in Cyber Domain

For the unknown network attacks and spying, feature mining is performed by PacketINT, and the mined features are associated with the massively captured packets to achieve the "wide-area defense" goal. This solution was published in Honeycon

Fraud Detection

Banking and e-commerce, when conducting online transactions, not only require account authentication information such as account number, password or two-factor authentication, but also need to use more information for anomaly detection. PacketINT's capability can be tapped. Deep features to participate in the fraud decision algorithm.
